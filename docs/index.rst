Welcome to Microbe Visualizer's documentation!
==============================================
This project, designed and developed primarily by `Jacob Bieker <https://www.jacobbieker.com/>`_, was created to help
with visualizing biological imaging data, mostly data taken of zebrafish with light-sheet microscopy technique. This software
was developed as part of ParthasarthyLab.
